# Инструкция по использованию админ-панели

## Вход в систему

1. Перейдите по адресу: `https://ваш-сайт.vercel.app/admin`
2. Нажмите "Login with Netlify Identity"
3. Введите логин и пароль

## Редактирование контента

### Темы курса

- В левом меню выберите "Темы курса"
- Нажмите на существующую тему или создайте новую
- Заполните все поля:
  - Номер темы (1-26)
  - Название темы
  - Краткое описание
  - Основной текст (используйте Markdown)
  - Изображение (опционально)

### Markdown редактор

Поддерживаемые элементы:

- Заголовки: #, ##, ###
- **Жирный текст**: `**текст**`
- _Курсив_: `*текст*`
- Списки: `- элемент` или `1. элемент`
- Ссылки: `[текст](URL)`
- Изображения: `![альт](URL)`
- Таблицы: используйте символы `|`

### Загрузка файлов

1. Нажмите на иконку изображения в редакторе
2. Выберите "Загрузить файл"
3. Перетащите файл или выберите с компьютера
4. Файлы сохраняются в `/public/uploads/`

## Сохранение изменений

1. Нажмите "Save" в верхнем правом углу
2. Изменения автоматически коммитятся в GitHub
3. Сайт обновляется через 1-2 минуты
